# Next-Gen Learning Dashboard

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- Lucide React

## Features
- Server-side data fetching
- Dynamic course cards
- Animated progress indicators
- Bento Grid dashboard
- Responsive design
- Loading skeletons
- Hover interactions

## Setup

npm install
npm run dev
